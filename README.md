# Controle de Gastos (React + TypeScript + Vite)
Aplicação simples para controle de despesas pessoais, construída com React, TypeScript e Vite. Este projeto é voltado para aprendizado e serve como base para gerenciar gastos, categorizar despesas e visualizar um resumo mensal.

**Funcionalidades principais**
- Adicionar novas despesas com título, valor, categoria e data.
- Visualizar lista de despesas por mês.
- Selecionar mês para filtragem.
- Resumo com total gasto e cards de resumo.
- Modal para adicionar despesas.
- Projeto responsivo e tipado com TypeScript.

**Póssiveis atualizações**
- Hoje o site salva pelos dados do navegador, criar uma base de dados para salvar úsuario, ou fazer com o importação e exportação de excel 
- Adicionar sistema de ganhos também, como salário etc.

**Tecnologias**
- React
- TypeScript
- Vite
- CSS

**Pré-requisitos**
- Node.js (v16+ recomendado)
- npm ou yarn

**Instalação**
1. Clone o repositório

```bash
git clone <REPO_URL>
cd controle-gastos
```

2. Instale dependências

```bash
npm install
# ou
yarn install
```

Abra http://localhost:5173 (ou a porta indicada) no navegador.
